36  /*** This class implements simplex-based direct search optimization
37   * algorithms.
38  
39   * <p>Direct search methods only use cost function values, they don't
40   * need derivatives and don't either try to compute approximation of
41   * the derivatives. According to a 1996 paper by Margaret H. Wright
42   * (<a href="http://cm.bell-labs.com/cm/cs/doc/96/4-02.ps.gz">Direct
43   * Search Methods: Once Scorned, Now Respectable</a>), they are used
44   * when either the computation of the derivative is impossible (noisy
45   * functions, unpredictable dicontinuities) or difficult (complexity,
46   * computation cost). In the first cases, rather than an optimum, a
47   * <em>not too bad</em> point is desired. In the latter cases, an
48   * optimum is desired but cannot be reasonably found. In all cases
49   * direct search methods can be useful.</p>
50  
51   * <p>Simplex-based direct search methods are based on comparison of
52   * the cost function values at the vertices of a simplex (which is a
53   * set of n+1 points in dimension n) that is updated by the algorithms
54   * steps.</p>
55  
56   * <p>The instances can be built either in single-start or in
57   * multi-start mode. Multi-start is a traditional way to try to avoid
58   * beeing trapped in a local minimum and miss the global minimum of a
59   * function. It can also be used to verify the convergence of an
60   * algorithm. In multi-start mode, the {@link #minimizes(CostFunction,
61   * int, ConvergenceChecker, double[], double[]) minimizes}
62   * method returns the best minimum found after all starts, and the
63   * {@link #getMinima getMinima} method can be used to retrieve all
64   * minima from all starts (including the one already provided by the
65   * {@link #minimizes(CostFunction, int, ConvergenceChecker, double[],
66   * double[]) minimizes} method).</p>
67  
68   * <p>This class is the base class performing the boilerplate simplex
69   * initialization and handling. The simplex update by itself is
70   * performed by the derived classes according to the implemented
71   * algorithms.</p>
72  
73   * @version $Id: DirectSearchOptimizer.java 1705 2006-09-17 19:57:39Z luc $
74   * @see CostFunction
75   * @see NelderMead
76   * @see MultiDirectional
77   */

86       * <p>The initial simplex is built from two vertices that are
87       * considered to represent two opposite vertices of a box parallel
88       * to the canonical axes of the space. The simplex is the subset of
89       * vertices encountered while going from vertexA to vertexB
90       * travelling along the box edges only. This can be seen as a scaled
91       * regular simplex using the projected separation between the given
92       * points as the scaling factor along each coordinate axis.</p>
93       * <p>The optimization is performed in single-start mode.</p>
94       * @param f cost function
95       * @param maxEvaluations maximal number of function calls for each
96       * start (note that the number will be checked <em>after</em>
97       * complete simplices have been evaluated, this means that in some
98       * cases this number will be exceeded by a few units, depending on
99       * the dimension of the problem)
100      * @param checker object to use to check for convergence
101      * @param vertexA first vertex
102      * @param vertexB last vertex
103      * @return the point/cost pairs giving the minimal cost
104      * @exception CostException if the cost function throws one during
105      * the search
106      * @exception NoConvergenceException if none of the starts did
107      * converge (it is not thrown if at least one start did converge)
108      */

123     /*** Minimizes a cost function.
124      * <p>The initial simplex is built from two vertices that are
125      * considered to represent two opposite vertices of a box parallel
126      * to the canonical axes of the space. The simplex is the subset of
127      * vertices encountered while going from vertexA to vertexB
128      * travelling along the box edges only. This can be seen as a scaled
129      * regular simplex using the projected separation between the given
130      * points as the scaling factor along each coordinate axis.</p>
131      * <p>The optimization is performed in multi-start mode.</p>
132      * @param f cost function
133      * @param maxEvaluations maximal number of function calls for each
134      * start (note that the number will be checked <em>after</em>
135      * complete simplices have been evaluated, this means that in some
136      * cases this number will be exceeded by a few units, depending on
137      * the dimension of the problem)
138      * @param checker object to use to check for convergence
139      * @param vertexA first vertex
140      * @param vertexB last vertex
141      * @param starts number of starts to perform (including the
142      * first one), multi-start is disabled if value is less than or
143      * equal to 1
144      * @param seed seed for the random vector generator
145      * @return the point/cost pairs giving the minimal cost
146      * @exception CostException if the cost function throws one during
147      * the search
148      * @exception NoConvergenceException if none of the starts did
149      * converge (it is not thrown if at least one start did converge)
150      */
151     public PointCostPair minimizes(CostFunction f, int maxEvaluations,
152                                    ConvergenceChecker checker,
153                                    double[] vertexA, double[] vertexB,
154                                    int starts, long seed)
155     throws CostException, ConvergenceException {
156 
157         // set up the simplex travelling around the box
158         buildSimplex(vertexA, vertexB);
159 
160         // we consider the simplex could have been produced by a generator
161         // having its mean value at the center of the box, the standard
162         // deviation along each axe beeing the corresponding half size
163         double[] mean              = new double[vertexA.length];
164         double[] standardDeviation = new double[vertexA.length];
165         for (int i = 0; i < vertexA.length; ++i) {
166             mean[i]              = 0.5 * (vertexA[i] + vertexB[i]);
167             standardDeviation[i] = 0.5 * Math.abs(vertexA[i] - vertexB[i]);
168         }
169 
170         RandomGenerator rg = new JDKRandomGenerator();
171         rg.setSeed(seed);
172         UniformRandomGenerator urg = new UniformRandomGenerator(rg);
173         RandomVectorGenerator rvg =
174             new UncorrelatedRandomVectorGenerator(mean, standardDeviation, urg);
175         setMultiStart(starts, rvg);
176 
177         // compute minimum
178         return minimizes(f, maxEvaluations, checker);
179 
180     }

213     /*** Minimizes a cost function.
214      * <p>The simplex is built from all its vertices.</p>
215      * <p>The optimization is performed in multi-start mode.</p>
216      * @param f cost function
217      * @param maxEvaluations maximal number of function calls for each
218      * start (note that the number will be checked <em>after</em>
219      * complete simplices have been evaluated, this means that in some
220      * cases this number will be exceeded by a few units, depending on
221      * the dimension of the problem)
222      * @param checker object to use to check for convergence
223      * @param vertices array containing all vertices of the simplex
224      * @param starts number of starts to perform (including the
225      * first one), multi-start is disabled if value is less than or
226      * equal to 1
227      * @param seed seed for the random vector generator
228      * @return the point/cost pairs giving the minimal cost
229      * @exception NotPositiveDefiniteMatrixException if the vertices
230      * array is degenerated
231      * @exception CostException if the cost function throws one during
232      * the search
233      * @exception ConvergenceException if none of the starts did
234      * converge (it is not thrown if at least one start did converge)
235      */
236     public PointCostPair minimizes(CostFunction f, int maxEvaluations,
237                                    ConvergenceChecker checker,
238                                    double[][] vertices,
239                                    int starts, long seed)
240     throws NotPositiveDefiniteMatrixException,
241     CostException, ConvergenceException {
242 
243         try {
244             // store the points into the simplex
245             buildSimplex(vertices);
246 
247             // compute the statistical properties of the simplex points
248             VectorialMean meanStat = new VectorialMean(vertices[0].length);
249             VectorialCovariance covStat = new VectorialCovariance(vertices[0].length);
250             for (int i = 0; i < vertices.length; ++i) {
251                 meanStat.increment(vertices[i]);
252                 covStat.increment(vertices[i]);
253             }
254             double[] mean = meanStat.getResult();
255             RealMatrix covariance = covStat.getResult();
256             
257 
258             RandomGenerator rg = new JDKRandomGenerator();
259             rg.setSeed(seed);
260             RandomVectorGenerator rvg =
261                 new CorrelatedRandomVectorGenerator(mean,
262                                                     covariance, 1.0e-12 * covariance.getNorm(),
263                                                     new UniformRandomGenerator(rg));
264             setMultiStart(starts, rvg);
265 
266             // compute minimum
267             return minimizes(f, maxEvaluations, checker);
268 
269         } catch (DimensionMismatchException dme) {
270             // this should not happen
271             throw new RuntimeException("internal error");
272         }
273 
274     }

342     /*** Build a simplex from two extreme vertices.
343      * <p>The two vertices are considered to represent two opposite
344      * vertices of a box parallel to the canonical axes of the
345      * space. The simplex is the subset of vertices encountered while
346      * going from vertexA to vertexB travelling along the box edges
347      * only. This can be seen as a scaled regular simplex using the
348      * projected separation between the given points as the scaling
349      * factor along each coordinate axis.</p>
350      * @param vertexA first vertex
351      * @param vertexB last vertex
352      */
353     private void buildSimplex(double[] vertexA, double[] vertexB) {
354 
355         int n = vertexA.length;
356         simplex = new PointCostPair[n + 1];
357 
358         // set up the simplex travelling around the box
359         for (int i = 0; i <= n; ++i) {
360             double[] vertex = new double[n];
361             if (i > 0) {
362                 System.arraycopy(vertexB, 0, vertex, 0, i);
363             }
364             if (i < n) {
365                 System.arraycopy(vertexA, i, vertex, i, n - i);
366             }
367             simplex[i] = new PointCostPair(vertex, Double.NaN);
368         }
369 
370     }

372     /*** Build a simplex from all its points.
373      * @param vertices array containing all vertices of the simplex
374      */
375     private void buildSimplex(double[][] vertices) {
376         int n = vertices.length - 1;
377         simplex = new PointCostPair[n + 1];
378         for (int i = 0; i <= n; ++i) {
379             simplex[i] = new PointCostPair(vertices[i], Double.NaN);
380         }

383     /*** Build a simplex randomly.
384      * @param generator random vector generator
385      */
386     private void buildSimplex(RandomVectorGenerator generator) {
387 
388         // use first vector size to compute the number of points
389         double[] vertex = generator.nextVector();
390         int n = vertex.length;
391         simplex = new PointCostPair[n + 1];
392         simplex[0] = new PointCostPair(vertex, Double.NaN);
393 
394         // fill up the vertex
395         for (int i = 1; i <= n; ++i) {
396             simplex[i] = new PointCostPair(generator.nextVector(), Double.NaN);
397         }

454     /*** Minimizes a cost function.
455      * @param f cost function
456      * @param maxEvaluations maximal number of function calls for each
457      * start (note that the number will be checked <em>after</em>
458      * complete simplices have been evaluated, this means that in some
459      * cases this number will be exceeded by a few units, depending on
460      * the dimension of the problem)
461      * @param checker object to use to check for convergence
462      * @return the point/cost pairs giving the minimal cost
463      * @exception CostException if the cost function throws one during
464      * the search
465      * @exception ConvergenceException if none of the starts did
466      * converge (it is not thrown if at least one start did converge)
467      */
468     private PointCostPair minimizes(CostFunction f, int maxEvaluations,
469                                     ConvergenceChecker checker)
470     throws CostException, ConvergenceException {
471 
472         this.f = f;
473         minima = new PointCostPair[starts];
474 
475         // multi-start loop
476         for (int i = 0; i < starts; ++i) {
477 
478             evaluations = 0;
479             evaluateSimplex();
480 
481             for (boolean loop = true; loop;) {
482                 if (checker.converged(simplex)) {
483                     // we have found a minimum
484                     minima[i] = simplex[0];
485                     loop = false;
486                 } else if (evaluations >= maxEvaluations) {
487                     // this start did not converge, try a new one
488                     minima[i] = null;
489                     loop = false;
490                 } else {
491                     iterateSimplex();
492                 }
493             }
494 
495             if (i < (starts - 1)) {
496                 // restart
497                 buildSimplex(generator);
498             }
499 
500         }
501 
502         // sort the minima from lowest cost to highest cost, followed by
503         // null elements
504         Arrays.sort(minima, pointCostPairComparator);
505 
506         // return the found point given the lowest cost
507         if (minima[0] == null) {
508             throw new ConvergenceException("none of the {0} start points"
509                     + " lead to convergence",
510                     new String[] {
511                             Integer.toString(starts)
512                     });
513         }
514         return minima[0];
515 
516     }

536     /*** Evaluate all the non-evaluated points of the simplex.
537      * @exception CostException if no cost can be computed for the parameters
538      */
539     protected void evaluateSimplex()
540     throws CostException {
541 
542         // evaluate the cost at all non-evaluated simplex points
543         for (int i = 0; i < simplex.length; ++i) {
544             PointCostPair pair = simplex[i];
545             if (Double.isNaN(pair.cost)) {
546                 simplex[i] = new PointCostPair(pair.point, evaluateCost(pair.point));
547             }
548         }
549 
550         // sort the simplex from lowest cost to highest cost
551         Arrays.sort(simplex, pointCostPairComparator);
552 
553     }

555     /*** Replace the worst point of the simplex by a new point.
556      * @param pointCostPair point to insert
557      */
558     protected void replaceWorstPoint(PointCostPair pointCostPair) {
559         int n = simplex.length - 1;
560         for (int i = 0; i < n; ++i) {
561             if (simplex[i].cost > pointCostPair.cost) {
562                 PointCostPair tmp = simplex[i];
563                 simplex[i]        = pointCostPair;
564                 pointCostPair     = tmp;
565             }
566         }
567         simplex[n] = pointCostPair;
568     }

570     /*** Comparator for {@link PointCostPair PointCostPair} objects. */
571     private static Comparator pointCostPairComparator = new Comparator() {
572         public int compare(Object o1, Object o2) {
573             if (o1 == null) {
574                 return (o2 == null) ? 0 : +1;
575             } else if (o2 == null) {
576                 return -1;
577             } else {
578                 double cost1 = ((PointCostPair) o1).cost;
579                 double cost2 = ((PointCostPair) o2).cost;
580                 return (cost1 < cost2) ? -1 : ((o1 == o2) ? 0 : +1);
581             }
582         }
583     };
